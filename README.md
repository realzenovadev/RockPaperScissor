# Rock Paper Scissors (Python)

This is a simple Rock Paper Scissors game that I built using Python.  
The game runs in the terminal and lets the user play against the computer.

## Features
- User vs computer gameplay
- Random computer moves
- Input validation
- Score counting
- Option to play again

## How to Run
1. Make sure Python is installed.
2. Run the file using:python main.py
 
## Technologies Used
- Python
- random module

## What I Learned
- How to use functions to organize code
- Taking and validating user input
- Using dictionaries to make output readable
- Writing basic game logic in Python
